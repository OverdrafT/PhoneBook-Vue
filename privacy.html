Privacy Policy

1. General Information

This Privacy Policy describes how your personal information is handled in this PhoneBook web application. This project is created for educational purposes within the "Web Programming" course using the MVC pattern.

2. Data Collection

We collect information that you voluntarily provide through the registration and profile forms, specifically:

  • Full Name
  • Email Address
  • Gender
  • Date of Birth
3. Data Usage & Storage

Unlike simple static sites, this application uses Web Storage API (localStorage) to persist your data locally within your browser. No data is transmitted to any remote server or external database. You can clear your data at any time by clearing your browser's local storage or cache.

4. Technical Implementation

The application is built using pure JavaScript (Vanilla JS), following a modular architecture to demonstrate logic separation between data (Model), interface (View), and logic (Controller).
